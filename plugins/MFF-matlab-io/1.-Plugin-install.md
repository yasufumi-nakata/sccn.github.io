EEGLABメニュー項目を選択 **ファイル>インポートデータ> EEGLABの機能とプラグインの使用 > EGI/Phillips .mff ファイルから** 以下が表示されます。

![](EEGLAB_install2.png)

MFFmatlabioプラグインがインストールされていない場合、EEGLABはインストールを促します(参照: [プラグインのインストール](1.-Plugin-install.html))。この後、MFFmatlabioは、以下に示すようにMFF再資源を選択するためのインターフェイスをプロンプトで表示します。

![](mff_import.png)
