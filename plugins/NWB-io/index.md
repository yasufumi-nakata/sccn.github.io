# NWBio EEGLABプラグイン

このプラグインは、注文なしでNeurodataからデータをインポートします
データ形式。 時間シリーズ、チャンネル情報、 
イベント情報は輸入されます。 EEGLABのインポート/エクスポートメニューを使用して、
インポート/エクスポートファイル、またはコマンドライン関数 pop_nwbimport を使用します。 m 点
または pop_nwbexport.m

# 事例紹介

チュートリアルEEGLABデータをエクスポートします。

```matlab
pop_nwbexport(EEG, 'test.nwb');
```

上記のファイルをインポートします。

```matlab
EEG = pop_nwbimport('test.nwb');
```

# EEGLABの利用

メニュー項目「File」>「EEGLAB」拡張子**を使用し、「nwbio」プラグインを検索します。 見つけたら
インストールします。 次に、メニュー項目「**File」を使用します。 インポートデータ > NWBファイル**からファイルをインポートします。 お問い合わせ
ファイルを選択すると、次のグラフィカルインターフェイスがポップアップ表示されます。

![Screenshot 2024-04-13 に 3 12 11 午後](https://github.com/sccn/nwbio/assets/1872705/a25b146c-f235-4e39-ac7d-979a29e45796)

EEGLAB(EEGLABイベントタイプ)の主要情報として使用するイベント情報を決定します。 
EEGLABは、他のフィールドを使うこともできますが、イベントタイプでほとんどの操作を行います。 すべてのフィールドに注意
EEGLABにインポートされますが、プライマリフィールドは重複し、イベントタイプフィールドで表示されます。

チェックボックスは、データファイル(グレーアウト時)に存在するEEGLABイベントとしてニューロンのスパイクをインポートすることができます。
スパイクは存在しません。 EEGLABは、全てのNWB情報をインポートしません(空間の動き、カルシウムイメージングなど)。
この EEGLAB プラグインは、電気生理学データのみをインポートします。

EEGまたはiEEGデータをエクスポートするには、メニュー項目**File > Export >を使用してください。 NWBファイルへのデータ** 次のグラフィカルインターフェイス
NWB固有の情報を入力できるようにポップアップします。 

![Screenshot 2024-04-13 に 3 25 03 PM](https://github.com/sccn/nwbio/assets/1872705/aa287c74-0e2e-4576-89ea-ccabc939ab21)

ほとんどのエントリは NWB 固有のので、参照することができます [NWB ドキュメント](https://www.nwb.org/) 詳細については、. 最後の2 
エントリーはEEGLABに必須です。 「エクスポートするイベントフィールド」は、EEGLABイベントフィールドをエクスポートできるリストです。 イベントのオンセットと期間
デフォルトでエクスポートされ、リストされていない。 "Export channel (x, y, z) の場所" でチャンネルの場所をエクスポートできます。 ご利用の場合 
テンプレートチャンネルの場所は、このオプションがチェックされていないままにすることができます。
