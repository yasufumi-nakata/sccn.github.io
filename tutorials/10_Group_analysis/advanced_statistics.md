最適化方法の違い 
- 通常のイースト スクエア(WLS): 最もシンプルで最速な方法です。 MATLAB の glmfit 機能と同じです。
- 重くされたイースト スクエア(WLS): デフォルト。 このメソッドは、自分のアウターの尤度に基づいて、個々の試験にいくつかの重量を属性します。
- 反重されたLeastの正方形(IRLS): デフォルト。 このメソッドは、自分のアウターの尤度に基づいて、個々の試験にいくつかの重量を属性します。
 
LIMOファク。
------
これらの質問はArnaud Delormeによってコンパイルされました。 ほとんどの回答は、Cyril Pernet、LIMOの主な著者です。

* **質問: ブーツトラップは2ndレベルのみ、または1stレベルと1つの被写体を見るとできますか?** ブートストラップの統計は2番目のレベルでのみ行われます。

* **質問: ANOVAの分析では、第2レベルのポストホックだけとは対照的ですか?** いいえ、コントラストは最初のレベルでも計算できます。

* **質問: (A) は 2 つのレベル 1 ベータ パラメータ (条件 1 と 2) の間のグループ (2nd) レベルでペア化された t-test を計算し、 (B) は条件 1 と 2 を (1st) の単一の対象レベル (2 ベータの差) との間のコントラストを計算し、その違いのグループ レベルで 1 つのサンプル t-test を計算する?** 理論的には、これは等価です。 しかし、LIMOは堅牢な統計を使用し、グループレベルでYuen t-testを使用するため、グループレベルでのt-testを使用するのが望ましいです。

* **質問: できます LIMOの被写体の2グループ以上を処理するか?** 2番目のレベルの統計では、2つのネストグループを一度に行うことはできません。

* **質問: 第1レベルの変数の数の点で制限は?** 制限はありません。 あなたが望むように、多くのカテゴリと連続変数を持つことができます。 

* **質問: 重量付きイーストスクエア(WLS)の最適化を1stレベルで使用したときにエラーが発生することがありますか?** WLSでは、サンプルよりも多くの試用版を持っている必要があります。 サンプルの数を減らすために時間ウィンドウまたは周波数範囲を減らすことができます。 これは、通常のイーストスクエアの問題ではありません。 IRLS(イテレーション・リウェイト・イースト・スクエア)は、WLSと比較しても遅く、ゲインが最小限に抑えられているため、ベストな方法ですが、練習では使用できない可能性があります。

* **質問: セッションごとに2回、セッションごとに連続したvar(questionnaire)があります。 セッションと連続varとのやり取りを計算するにはどうすればよいですか?** 繰り返し測定 ANOVA をコバリエートで使用したいのですが、今は LIMO では使用していません(maybe check old_rep_ANOVA.m)。 ANCOVA は使えますが、セッションが被写体にペアリングされているという事実を失います。 ANCOVA では、インタラクション用の製品で追加の列を含むテキストファイルを格納したいです。 2つのセッションとx1〜x4の連続変数がある場合、最後の列には(-x1、-x2、x3、x4)が含まれています。 理論では、2つの分類変数、1つの連続変数、1つの相互作用を持つテキストファイルがあります。 例えば、x1=1、x2=2、x3=3、x4=4 の場合、テキストファイルには
```
1  0  1  -1
1  0  2  -2
0  1  3  3
0  1  4  4
```
> しかし、実際には、どの変数が分類され、ユーザーが要求せずに連続しているかを知ることは困難であるため、ベータの2つのリスト、セッション1とセッション2の1のための1、およびあなたのコワリエートファイルが含まれている必要があります。
```
1  -1
2  -2
3  3
4  4
```
> 連続変数が最初のレベル(反応時間など)にある場合、関数 [リモ_split_continuous.m](https://github.com/LIMO-EEG-Toolbox/limo_tools/blob/master/limo_split_continuous.m) 1st レベルで各分類変数に対して連続変数を分割するためにも使用できます。 そのため、連続変数が1つある場合は、セッション1とセッション2の1つ(これは、EEGLABの最初のLIMOメニューから行うことができます)の2つのベータパラメータが取得されます。 その後、セッションと連続変数間の相互作用の用語は、 ANCOVA を使用して 2 番目のレベルで計算することができます。

* **質問: 私は、刺激物の50%がターゲット刺激であり、他の50%が散布者であるGo-nogoタスクで反応時間を持っています(抽出物は、ディストラクターに応答しない)。 正しい反応を分析したい(目標試験の応答とノーゴーノンターゲット試験の応答なし)。 相手の反応時間がないにもかかわらず、コワリエートとして反応時間を含める方法はありますか?** 可能です。 これは、試験全体でレベル1の分析になります。 ディストラクターの場合、反応時間を0に設定します。つまり、これらの試験は、コワリエート分析のために反応時間と無視されます。 しかし、解散者に対して反応時間がないため、反応時間と試用タイプの相互作用を計算できません。 2番目のレベルでは、試用型(ターゲット対ディストラクター)と1サンプルt-testを反応時間ベータパラメータで実行できます。

* **質問: 9種類の視覚刺激、顔の3種類、各刺激に対する3回繰り返しがあります(代表1、2、3)。 顔の違いを見てみたい。 LIMOでそれを行う最良の方法は何ですか?** 顔の型(反復の比例)を含む EEGLAB に変数がある場合、最初のレベルでそれを使用することができ、2 番目のレベルでの 3 種類の顔の間の ANOVA を実行します。 しかし、より良い解決策は、最初のレベル(合計9ベータを獲得するために)でフェースタイプと繰り返しの両方を入力することです。その後、3つの顔の3つの繰り返しに対してベータ(平均を計算する)をグループ化し、3つのコントラストを比較します。 EEGLABでは、STUDYデザインを作成するときに顔の型をグループ化し、コントラストは自動的に作成されます。 このソリューションの利点は、モデルに反復が含まれているため、繰り返しによる信号のバリエーションが考慮されます。 第3のソリューションは、第1レベルモデル(9ベータ)の繰り返しと直面し、第2レベル(顔×反復)で2方向のANOVAを実行し、顔の主な効果を調べることです。 しかし、反復に興味を持たない場合、このソリューションは潜水的であり、双方向のANOVAが1方向と比較して統計力が低下するからです。
