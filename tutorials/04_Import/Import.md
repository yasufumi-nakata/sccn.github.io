データのインポートは、多くの場合、EEG分析で開始する最初の最も重要なステップです。 様々なデータフォーマット、チャンネル位置ファイル、EEGデータと同期する必要があるイベントデータファイルなど、チャレンジングに繋がることができます。 この一連のチュートリアルでは、EEGデータと補足ファイルをインポートすることであなたを導きます。
